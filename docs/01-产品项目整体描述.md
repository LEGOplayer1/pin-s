# 纸间（ZhiJian）· 产品项目整体描述

> 「纸间」是一款面向 Windows（优先）的桌面便利贴应用，以"精致极简主义"为核心设计语言，
> 提供轻盈、安静、不打扰的记录体验。

---

## 1. 产品定位与目标

**一句话定位**  
> 桌面上的一张轻量便签纸。

**产品愿景**  
成为用户打开电脑就会顺手写上几笔的"第二桌面"——比系统自带 Sticky Notes 更克制、更美；比笔记软件（Notion、Obsidian）更轻量。

**目标用户**
- 重度电脑用户：需要临时记录灵感、待办、代码片段
- 对视觉与质感敏感的设计师 / 内容创作者
- 希望桌面保持整洁但需要可视化便签的办公人群

**差异化（相比 Windows Sticky Notes）**
- 莫兰迪色系 + 霞鹜文楷，视觉更有温度
- 每张便利贴独立窗口，可自由排布、置顶、穿透
- 更低的内存占用与安装体积（Tauri 架构）
- 支持 Markdown 轻量格式，而不是纯文本

## 3. 功能清单（按优先级）

### P0 — MVP（必须）
| 编号 | 功能 | 说明 |
|---|---|---|
| F01 | 独立窗口便利贴 | 每张便利贴是一个独立桌面窗口，260×240 默认尺寸 |
| F02 | 自由拖拽 | 顶部标题栏区域即拖拽区 |
| F03 | 自由缩放 | 右下角角标拖拽调整尺寸 |
| F04 | 6 色切换 | 米白 / 雾蓝 / 淡粉 / 草绿 / 薰衣草 / 鹅黄 |
| F05 | 内容编辑 | 可编辑富文本；失焦自动渲染 Markdown 子集 |
| F06 | 置顶（Always on Top） | 一张便签可钉在所有窗口前面 |
| F07 | 关闭 / 新建 | 右上角 ✕ 关闭；+ 号新建一张（默认同色） |
| F08 | 系统托盘 | 右键菜单：新建 / 显示全部 / 最小化全部 / 退出 |
| F09 | 单实例运行 | 再次启动不重复打开主进程，仅新增便利贴 |
| F10 | 本地持久化 | 便签内容、颜色、位置、尺寸本地保存 |
| F11 | 快捷键 | `Ctrl + N` 新建，`Ctrl + W` 关闭当前 |

### P1 — 增强（0.2 版本）
| 编号 | 功能 | 说明 |
|---|---|---|
| F12 | 搜索 / 过滤 | 托盘搜索框，快速在便签中检索关键词 |
| F13 | 提醒时间 | 为便签设置闹钟，到点托盘通知 |
| F14 | 自动整理布局 | 一键把散落便签排列成网格/瀑布流 |
| F15 | 聚焦模式 | 点击某便签时，其他便签虚化淡出 |
| F16 | 鼠标穿透 | 勾选后便签成为"桌面水印"，不阻挡操作 |
| F17 | 开机自启 | 登录系统即启动并恢复上次便签 |

### P2 — 长期演进
| 编号 | 功能 | 说明 |
|---|---|---|
| F18 | 云同步 | 使用用户自托管 / WebDAV，跨设备同步 |
| F19 | 便签模板 | 待办清单 / 阅读笔记 / 灵感卡片 / 会议纪要 |
| F20 | 多语言 | 中文 / English |
| F21 | macOS & Linux 版本 | 跨平台 |

## 4. 设计语言

**色彩系统（莫兰迪低饱和）**

| token | hex | 语义 |
|---|---|---|
| `--note-cream`    | #FAF6EE | 默认，温暖米色 |
| `--note-blue`     | #E8EDF2 | 雾蓝，偏冷静 |
| `--note-pink`     | #F2E8E8 | 淡粉，温柔 |
| `--note-green`    | #E8F0E8 | 草绿，自然 |
| `--note-lavender` | #ECE8F0 | 薰衣草，浪漫 |
| `--note-yellow`   | #F5F0DC | 鹅黄，明亮 |

辅助文字颜色：
- 正文 `#4A433A`（深赭褐）
- 次要文字 `#8A8279`（雾灰）
- 时间戳等静音信息 `#B0A899`

**字体**
- 中文：霞鹜文楷（LXGW WenKai）
- 英文标题：Playfair Display
- 英文正文：Crimson Pro
- 回退：系统默认中文字体 + serif

**视觉质感**
- 便利贴主体：纯色 + 极细微噪点纹理（SVG 内联）
- 阴影：多层柔光阴影，模拟真实纸张
- 窗口：无边框、背景透明、可调节尺寸
- 标题栏：比便签本体深一个色阶，强化卡片感

## 5. 界面线框说明

每张便利贴窗口结构（自顶向下）：

```
┌──────────────────────────────┐
│  ○ ○ ○ ○ ○ ○    📌  +  ✕    │  ← 标题栏（可拖拽）
│  ──────────────────────────  │
│  # 欢迎使用纸间               │
│                              │
│  这是一段便签内容，支持       │
│  **粗体**、*斜体*、`代码`。   │
│                              │
│  > 引用：像真实便签纸一样     │
│     贴在你的桌面上。          │
│                              │
│  纸间        6/10 09:15   ╲ │  ← 底部 + 右下角尺寸调整
└──────────────────────────────┘
```

标题栏左侧 6 颗色点：点击即切换便签颜色；  
标题栏右侧依次为置顶按钮 / 新建按钮 / 关闭按钮。

## 6. 非功能需求

| 指标 | 目标 |
|---|---|
| **安装包大小（Windows portable）** | ≤ 15 MB |
| **空窗内存占用** | ≤ 30 MB（主进程 + 单张便利贴） |
| **5 张便利贴总内存** | ≤ 60 MB |
| **冷启动** | ≤ 1 秒，视觉上接近即时 |
| **持久性** | 关机 → 重启后恢复全部便签的内容 / 颜色 / 位置 / 尺寸 |
| **离线可用** | 应用全部功能无需联网 |
| **安全** | 便利贴仅写入用户目录（`%APPDATA%`），不收集任何数据 |
